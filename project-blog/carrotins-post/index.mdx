## 🎤 Work Story

- ### 프로젝트 기간

  2020.01.01 ~ 2020.03.31

- ### 프로젝트 투입 배경

  1월 초에 open한 Carrot 보험 사이트 프로젝트에서 Open 후 _안정화 작업을 위해_ 1월 15일부터 투입 되어 Open 후 수정 사항들 및 신규 보험 상품 관련 퍼블리싱 및 추가 요건 사항들에 대해서 대응 후 철수.

- ### 프로젝트 이슈 대응 및 성과 내용

  구축팀 철수 후 신규 페이지 퍼블리싱

  > 투입 후 약 2주간에 인수인계 후 바로 2월에 초에 오픈하는 _퍼마일자동차 상품 컨텐츠 코딩 및 구축 퍼블리셔들이 끝내지 못한 수정 사항 처리,_ _오픈기념 이벤트 페이지 반응형 퍼블리싱 및, 모바일 메인 리뉴얼 화면 퍼블리싱_

  react 프로젝트를 하면서 예상하지 못한 문제들을 경험 하였고 이런 문제들을 통해서 앞으로 react 프로젝트에 처음부터 투입이 된다면 시작 부터 고려 해야되는 사항들에 대해서 알게 되었고 그리고 간단한 react에서 클릭 이벤트와 스크롤 이벤트 정도 까지는 할 수 있게 되었습니다.
