- ### 🗓️ 프로젝트 기간

  2018.04.16 ~ 2018.04.23

- ### 📌 프로젝트 개요

  두산 E-commerce **어드민 사이트**를 단독 퍼블리싱(약 5일)으로 구축.  
  병행 프로젝트 일정 속에서 **내부 어드민 템플릿 재사용** 전략으로 빠르게 납품.

- ### 🛠 주요 업무

  - 기존 어드민 템플릿(트래블 메이트 기반)을 **구조 재활용**하고 **CSS 커스터마이징**
  - **BEM 네이밍 적용**으로 스타일 구조화 및 유지보수성 확보
  - 페이지 레이아웃/폼/테이블 중심의 **어드민 UI 퍼블리싱**
  - jQuery 중심의 기본 인터랙션 보완

- ### 🎯 성과 및 배운 점

  - 템플릿 재사용과 구조화된 네이밍으로 **5일 내 단기 납품** 달성
  - 재사용 가능한 **어드민 퍼블리싱 템플릿**을 정립하여 이후 유사 프로젝트에 활용
