- ### 🗓️ 프로젝트 기간

  2024.04.01 ~ 2024.05.15

- ### 📌 프로젝트 개요

  LG 글로벌 사이트의 UI를 활용하여 독일 멀티샵 신규 사이트를 구축하기 위해 1.5개월 단기 계약으로 투입.  
  Pug 기반 퍼블리싱 환경에 적응하며 짧은 일정 내 요구된 화면 개발을 지원.

- ### 🛠 주요 업무

  - Pug 기반 퍼블리싱 환경 분석 및 적용
  - 신규 페이지 퍼블리싱 및 UI 구현
  - 퍼블리싱 산출물(HTML, SCSS, JS) 전달 프로세스 개선
  - 개발팀이 Pug 적용에 어려움을 제기함에 따라, **HTML 변환·전달 방식**을 협의 후 적용

- ### 🎯 성과 및 배운 점

  - 프로젝트 투입 후 Pug를 빠르게 학습·적용하여 일정 차질 없이 퍼블리싱 대응
  - **개발팀이 “퍼그 파일로 공유 시 적용이 어렵다”는** 문제를 제기함 →  
    이를 해결하기 위해 **Pug 파일을 HTML로 변환 + 수정 부분 주석 처리** 후 SCSS·JS와 함께 별도로 전달하는 프로세스 제안
  - 해당 방식으로 협업 문제를 해소하여 원활한 개발 진행을 지원
  - 2개월 이내 멀티샵(독일) 사이트 개발 일정에 맞춰 주요 화면 퍼블리싱 완료
  - 새로운 템플릿 엔진(Pug) 환경 적응 경험을 통해 대응력과 문제 해결 능력을 강화
