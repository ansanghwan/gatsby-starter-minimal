- ### 🗓️ 프로젝트 기간

  2016.08.05 ~ 2016.11.21

- ### 📌 프로젝트 개요

  EBS 방송 프로그램 홈페이지의 **기존 디자인 스킨(5종) 유지**를 전제로,  
  **신규 반응형 스킨을 추가 구현**하여 다양한 화면 크기에서의 접근성을 개선.

- ### 🛠 주요 업무

  - 기존 HTML 구조를 유지하고 **CSS 레이어만으로 반응형 대응** 설계
  - **신규 반응형 스킨 퍼블리싱** 및 스타일 최적화
  - 기획 누락 콘텐츠 발생 시 **디자인·기획 생략 후 실시간 수정 대응**
  - 반응형 레이아웃 테스트 및 크로스 브라우징 검수

- ### 🎯 성과 및 배운 점

  - 기존 스킨 호환성을 유지하면서 **신규 반응형 스킨 추가 완료**
  - 제한된 시간과 잦은 변경 속에서도 **일정 준수와 안정적 결과물 제공**
  - 기존 소스 수정 최소화 전략을 통해 **운영 안정성과 유지보수 효율성 확보**
