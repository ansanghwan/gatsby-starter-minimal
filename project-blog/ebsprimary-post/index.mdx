- ### 🗓️ 프로젝트 기간

  2017.04.05 ~ 2017.07.21

- ### 📌 프로젝트 개요

  EBS 초등 홈페이지의 **반응형 리뉴얼**을 위해 인수인계 후 운영과 병행 작업.  
  퇴사한 담당자의 공백을 메우고, 남은 주요 메뉴와 서브 페이지 퍼블리싱을 완수.

- ### 🛠 주요 업무

  - **문제은행, 교사·학부모, 통합검색, 이벤트, 고객센터** 메뉴의 서브 페이지 반응형 퍼블리싱
  - CSS 관리 효율을 높이기 위해 **모듈화(import 방식)**로 스타일 분리
  - 유지보수 편의성을 위한 **메뉴별·페이지별 주석 구조화**
  - PC·모바일 시안이 상이한 경우, 기획/개발과 협의해 **레이아웃 통일** 또는 **별도 레이아웃 처리**

- ### 🎯 성과 및 배운 점

  - 인수인계 후 빠르게 적응해 **남은 주요 메뉴 리뉴얼을 일정 내 완료**
  - 스타일 모듈화와 주석 설계로 **유지보수성과 가독성을 향상**
  - 반응형 개발에서의 **시안 불일치 이슈 대응** 경험을 통해 커뮤니케이션·문제 해결 역량 강화
