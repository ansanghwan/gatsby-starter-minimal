- ### 🗓️ 프로젝트 기간

  2020.11.16 ~ 2020.12.15

- ### 📌 프로젝트 개요

  T로밍 메뉴 오픈 이후, 접근성 고려가 부족했던 마크업/스크립트 이슈를 개선하기 위해 투입.  
  접근성 컨설팅사에서 제안한 수정 과제를 정리·적용하고, 실사용 관점의 보완 방안을 함께 제시.

- ### 🛠 주요 업무

  - 컨설팅 리포트 기반 **이슈 리스트업 및 수정 계획 수립**
  - **모달 접근성 개선**: 포커스 트랩/회귀, 선형 구조 보정, `role="dialog"`, `aria-modal`, 라벨링 정비
  - **달력 UI ARIA 적용**: 키보드 내비게이션, 상태 전달(`aria-selected`, `aria-current`)
  - **스위치·탭 UI 보정**: 역할/상태 속성(`role="tab"`, `aria-selected`, `aria-controls`) 정합화
  - **대체 텍스트 보완**: 이미지 `alt` 누락 구간 정리·적용
  - 퍼블리싱 리스트 작성 및 **운영 퍼블/개발팀 인수인계**, Git으로 변경 이력 관리

- ### 🎯 성과 및 배운 점

  - 접근성 과제를 체계적으로 적용하여 **키보드/스크린리더 사용성 향상**
  - 모달·탭·달력 등 **상호작용 컴포넌트의 패턴 표준화**로 후속 이슈 예방
