- ### 🗓️ 프로젝트 기간

  2022.09.19 ~ 2022.12.18

- ### 📌 프로젝트 개요

  기존 웹 퍼블리셔의 철수로 인한 공백을 메우기 위해 긴급 투입.  
  "브랜드" 카테고리의 상품 상세 페이지 퍼블리싱과 인터랙션 구현을 담당.

- ### 🛠 주요 업무

  - 브랜드 카테고리 **상품 상세 페이지 퍼블리싱** (총 4개 상품 페이지)
  - PC 시안만 제공된 상태에서 모바일 화면을 **퍼블리셔 주도 설계 및 구현**
  - GSAP을 활용한 **스크롤 기반 애니메이션 구현**
  - 웹 접근성을 고려한 마크업 및 UI 구성

- ### 🎯 성과 및 배운 점

  - 투입 당시 지연 중이던 **4개 상품 상세 페이지를 단기간에 퍼블리싱 완료**
  - 시안이 없는 모바일 환경을 스스로 설계·구현하며 **문제 해결 능력과 대응력 강화**
  - GSAP을 활용한 화려한 스크롤 인터랙션 구현 경험을 통해 **애니메이션 활용 스킬 확장**
  - 웹 접근성을 고려한 퍼블리싱으로 **품질과 사용자 경험을 동시에 확보**
