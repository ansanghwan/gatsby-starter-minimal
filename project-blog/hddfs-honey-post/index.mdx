- ### 🗓️ 프로젝트 기간

  2024.05.16 ~ 2024.11.01

- ### 📌 프로젝트 개요

  현대면세점 사이트의 _Honey 프로젝트_ 및 *인천 스마트 면세 구축*에 웹 퍼블리셔 PL로 투입되어, 신규 페이지 구축과 기존 페이지 개선 작업을 총괄.  
  운영팀 소스와의 충돌 가능성을 사전에 기획·개발팀과 협의하고 대응 방안을 마련하여, 안정적으로 프로젝트를 완료함.

- ### 🛠 주요 업무

  - 프로젝트 초기 퍼블리싱 환경 세팅 및 **공통 퍼블리싱 가이드 제작**
  - 신규 UI 화면 퍼블리싱 및 **기존 페이지 개선**
  - **운영팀과의 코드 충돌 방지 프로세스 제안 및 적용**
  - 팀 내 웹 퍼블리셔(3명) 업무 분담 및 일정 관리
  - 신규 UI 컴포넌트 마크업, 스크립트, 다국어 페이지 퍼블리싱
  - Honey 및 인천 스마트 면세 프로젝트 산출물 제작 및 인수인계

- ### 🎯 성과 및 배운 점

  - 초기 일정 지연 가능성(9월 오픈 목표)을 극복하기 위해 집중 근무, 2주 내 가이드와 신규 화면을 완성
  - 팀원에게 가이드를 공유하여 **재사용성 높은 퍼블리싱 구조 정착**
  - 프로젝트 종료 후 운영팀 소스와 병합 시 **이슈 없이 단일 머지 완료**
  - 다양한 UI 컴포넌트와 다국어 페이지 퍼블리싱 경험 축적
