- ### 🗓️ 프로젝트 기간

  2018.11.26 ~ 2019.01.10

- ### 📌 프로젝트 개요

  기존 포스코 사이트의 **‘제품’ 메뉴를 별도 제품 사이트로 분리 구축**하는 프로젝트에 투입.  
  **어드민 사이트 퍼블리싱**을 담당하고, 이후 프론트의 일부 화면을 지원.

- ### 🛠 주요 업무

  - **어드민 사이트 퍼블리싱**
    - 촉박한 일정으로 **디자인 시안 없이** 기획서를 기반으로 화면 퍼블리싱
  - **Bootstrap(Summernote) SCSS 커스텀**
    - 에디터 적용을 위해 Bootstrap SCSS 일부 커스텀
    - 자체 SCSS와 **클래스 충돌 방지 구조**로 설계
  - **프론트 페이지 일부 지원**
    - 프론트 사이트 **‘미디어’ 메뉴** 퍼블리싱

- ### 🎯 성과 및 배운 점

  - 시안 부재·단기 일정 환경에서도 **기획서 기반 퍼블리싱 프로세스 정립**으로 안정적 납품
  - Bootstrap 에디터 도입 시 **SCSS 충돌 최소화 전략** 수립 및 적용
  - 어드민/프론트 병행 경험으로 **운영·콘텐츠 양 측면의 UI 요구에 대한 대응력 강화**
