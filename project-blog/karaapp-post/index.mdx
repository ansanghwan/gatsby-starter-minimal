- ### 🗓️ 프로젝트 기간

  2021.07.26 ~ 2021.10.22

- ### 📌 프로젝트 개요

  기존 운영 담당자의 프로젝트 이동으로 인한 공백을 메우기 위해 긴급 투입.  
  Ionic 기반의 Kare App 운영과 부분 개선 퍼블리싱을 담당하여 안정적인 서비스 운영을 지원.

- ### 🛠 주요 업무

  - **운동 메뉴 추가**에 따른 신규 화면 퍼블리싱
  - **Kare App 메인 화면 퍼블리싱** 및 UI 보완
  - **브랜드 반응형 웹 페이지 퍼블리싱**
  - Ionic 하이브리드 환경에 맞춘 스타일/레이아웃 최적화

- ### 🎯 성과 및 배운 점

  - iOS 환경에서 특정 페이지 **하단 팝업 고정 이슈**를 해결
    - 원인: `safe-area-inset-bottom` 처리 미흡 → 안전 영역 고려한 레이아웃으로 수정
  - **달력 테이블의 세로 정렬 문제**(Ionic **Shadow DOM** 영향) 원인 파악 및 스타일링 보완
  - 단기간 내 여러 개선 과제를 수행하며 **하이브리드 앱 환경 대응력** 강화
