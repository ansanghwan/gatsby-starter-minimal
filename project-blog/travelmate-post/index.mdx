- ### 🗓️ 프로젝트 기간

  2018.01.15 ~ 2018.01.20

- ### 📌 프로젝트 개요

  여행용품 쇼핑몰 **Travelmate**의 어드민 초기 구축에 참여하여  
  **스타일 가이드 + 1차 퍼블리싱 버전**을 단기간에 완성, 개발팀에 전달.

- ### 🛠 주요 업무

  - **스타일 가이드 수립**(레이아웃, 타이포, 폼/테이블, 컴포넌트 규칙)
  - 어드민 **핵심 화면 1차 퍼블리싱** 및 산출물 구조 정리
  - 개발 진행 중 **추가 요청 화면 반영**(증분 퍼블리싱)
  - jQuery 기반 **공통 인터랙션 스크립트** 구현
  - 크로스 브라우징 대응(IE10+, Chrome, Firefox)

- ### 🎯 성과 및 배운 점

  - 가이드와 초기 산출물 제공으로 **개발팀 온보딩·진행 속도** 향상
  - 이후 유사 어드민 프로젝트에 재사용 가능한 **기초 템플릿** 축적
  - 퍼블리싱-개발 협업 플로우(초기 버전 → 증분 반영) 최적화
