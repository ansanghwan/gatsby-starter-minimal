## 🎤 Work Story

- ### 프로젝트 투입 배경

  kixxoil 반응형 리뉴얼 프로젝트에 PL 보다 일주일 먼저 투입 하여 프로젝트 진행 상황에 대하여 이야기를 듣고 작업 준비를 하였습니다.

  퍼블리셔는 저, PL 이렇게 2명으로 퍼블리싱 진행 하였습니다.

- ### 프로젝트 이슈 및 대응 사항

  1. 서브 페이지 퍼블리싱
     > 메인 페이지를 제외 한 전 페이지를 퍼블리싱을 진행 하였고 1차로 제가 퍼블리싱후 PL에게 넘겼고, PL이 전체적으로 재수정 하였습니다.<br/>
     > 페이지 퍼블리싱 완료 후 먼저 10월 말에 철수 하였습니다.
  2. 프로젝트 후기
     > 프로젝트를 항상 혼자 진행하다 처음으로 다른 사람과 협업을 한 프로젝트인대 프로젝트기간 동안 퍼블리셔 Pl과 갈등도 있었던 것 같습니다. <br/>
     > 모르는 사람들과 협업을 하게 되면 고려 해야될 점들을 알게 된 프로젝트 였으며, 리더가 프로젝트에 관련된 가이드 및 진행에 대해서 팀원과 이야기를 하지 않고 리드를 하지 않으면 어떤 결과를 초래 하는지 경험으로 알게 되었고 다음 내가 프로젝트 리더로 프로젝트를 진행 하게 된다면 이렇게 하면 안된 다는 것을 절실히 알게 된 프로젝트 였습니다.
